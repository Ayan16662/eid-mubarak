# 🌙 Eid Mubarak — عيد مبارك

A premium, award-quality **Eid Mubarak** website built with HTML5, CSS3, and Vanilla JavaScript.  
Featuring GSAP animations, canvas fireworks, Web Audio music, floating lanterns, and a glassmorphism design.

---

## 📁 File Structure

```
eid-mubarak/
├── index.html   ← Markup & structure
├── style.css    ← All styles, variables, animations
├── script.js    ← GSAP, cursor, stars, lanterns, fireworks, music
└── README.md
```

---

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `eid-mubarak`)
2. Upload all three files: `index.html`, `style.css`, `script.js`
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at:  
   `https://<your-username>.github.io/eid-mubarak/`
